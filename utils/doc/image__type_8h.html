Detailed Description

Contains a list of all frontend image type. Several members are deprecated in the current version.

Deprecated:

VERTICAL_CROSS and

HORIZONTAL_CROSS

images are no longer used as the profile_widget accounts for crosshair profiles as a more modular form of mean profile. When using image_types in a switch statement, use default: break; to circumvent warnings about missed members.
