void buffer_handler::getFrame ( )
slot

The main event loop of the buffer_handler class.

The function runs in an infinite loop until a stop is requested by setting running to false. Every microsecond, the values of the current_frame and old_frame are compared. If they are not equal, the current_frame is read from the file. This operation is memory locked to avoid simultaneous read and write operations on the frame array.
